Background

Using devices such as Jawbone Up, Nike FuelBand, and Fitbit it is now possible to collect a large amount of data about personal activity relatively inexpensively. These type of devices are part of the quantified self movement - a group of enthusiasts who take measurements about themselves regularly to improve their health, to find patterns in their behavior, or because they are tech geeks. One thing that people regularly do is quantify how much of a particular activity they do, but they rarely quantify how well they do it. In this project, your goal will be to use data from accelerometers on the belt, forearm, arm, and dumbell of 6 participants. They were asked to perform barbell lifts correctly and incorrectly in 5 different ways. More information is available from the website here: http://groupware.les.inf.puc-rio.br/har (see the section on the Weight Lifting Exercise Dataset).

Data Cleaning

The files are loaded mapping the NA, “” and “”#DIV/0!“” values into NA for missing values handling. The are a lot of NA values present in multiple variables. To preserve the space of this report, the summary() command output is omitted,

Remove the near zero variance predictors, to reduce the total size of the data frame

nzv_cols <- nearZeroVar(train[, -159])
if(length(nzv_cols) > 0) 
    train <- train[, -nzv_cols]
dim(train)
## [1] 19622   123

Now we have 123 predictors

The next step involves deleting the remaining columns that contains values with NA. These columns has almost all the values in NA, so we decided to delete the columns.

train = train[ , colSums(is.na(train)) == 0]
dim(train)
## [1] 19622    58

Finally, the data frame has 58 variables.

Generate the prediction files to submit to the autograder

pml_write_files = function(x){
    n = length(x)
    for(i in 1:n){
        filename = paste0("./prediction/problem_id_", i, ".txt")
        write.table(x[i], file = filename, quote = FALSE, row.names = FALSE, col.names = FALSE)
    }
}
pml_write_files(pred)
